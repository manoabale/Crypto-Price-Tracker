# Crypto-Price-Tracker


A simple cryptocurrency price tracker using the [CoinGecko API](https://www.coingecko.com/en/api).

## Features
- Search crypto by name (e.g. bitcoin, ethereum, dogecoin)
- Display current price, market cap, and 24h change
- Responsive design

## Setup
1. Clone the repo.
2. Open `index.html` in your browser.
3. Or deploy to GitHub Pages.

## Deployment
- Push this repo to GitHub.
- Go to **Settings → Pages → Source → main branch → /root**.
- Your app will be live at `https://your-username.github.io/crypto-price-tracker`.
